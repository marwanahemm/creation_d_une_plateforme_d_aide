
# 📋 Plateforme d'aide administratives

## 🖋️ Description

Plateforme d'aide permettant aux usagers d'effectuer leurs démarches administratives auprès des principaux organismes français : France Travail, Ameli et la CAF. Cette application web facilite l'accès aux informations et guide les utilisateurs pas à pas dans leurs démarches.

## ✨ Fonctionnalités

- 📱 **Responsive Design** : Interface accessible sur tous les appareils (mobile, tablette, desktop)
- 🔍 **Barre de recherche** : Trouvez rapidement la démarche dont vous avez besoin
- 📑 **Tutoriels détaillés** : Guides étape par étape pour chaque démarche administrative
- 🏢 **Multi-organismes** : Démarches pour France Travail, Ameli et la CAF
- 📊 **Niveau de difficulté** : Chaque démarche est classée (Facile, Moyen, Difficile)

## 🛠️ Technologies utilisées

- **Next.js 16.1.6** : Framework React pour applications web modernes
- **HTML5** : Langage de balisage pour la structure des pages
- **CSS3** : Stylisation et mise en page responsive
- **JavaScript ES6+** : Logique et interactivité de l'application
- **Tailwind CSS 4.1.18** : Fourniture de classes utilitaires réutilisables qui rendent le style rapide et efficace
- **Vercel** : Hébergement d'applications web modernes
- **Supabase** : Outil Open Source permettant aux développeurs d'accéder à une base de données
